# 可视化校园自习室预约系统

这是一个不依赖外部 npm 包的可运行前后端示例，按详细设计说明书实现了核心 MVP：

- 学生登录、房间筛选、座位可视化、在线预约
- 预约取消、扫码签到模拟、暂离、结束使用
- 超时未签到自动释放并生成违规记录
- 管理员资源查看、座位维修/禁用/恢复
- 预约规则配置、统计看板、审计日志

## 运行

```bash
npm start
```

打开：

```text
http://localhost:3000
```

## 数据存储

后端使用 `backend/data/db.json` 作为演示数据文件。首次启动会自动生成种子数据。

重置数据：

```bash
npm run reset-data
```
